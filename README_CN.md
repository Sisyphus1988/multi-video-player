# 本地视频多开播放器

一个轻量的本地多视频播放器，当前以单个 `HTML` 文件形式提供。

它适合在一个页面里同时打开、预览和控制多个本地视频，不依赖后端服务，也不需要额外安装项目依赖。

## 功能特性

- 支持一次选择多个本地视频文件
- 支持通过 `webkitdirectory` 选择整个文件夹
- 支持拖拽文件或文件夹到页面中导入
- 每个视频都可以独立播放、暂停、静音、调音量、调倍速、删除
- 支持批量播放、暂停、静音、取消静音、清空列表
- 支持全屏和画中画
- 支持倍速滑块和常用倍速预设
- 支持按文件名实时搜索
- 支持播放进度、缓冲进度、悬停时间提示
- 支持显示分辨率和文件大小等元信息
- 支持常见键盘快捷键
- 支持响应式网格布局，方便多开查看

## 技术栈

- HTML
- CSS
- 原生 JavaScript

## 使用方式

1. 用 Chromium 内核浏览器打开 `video-player.html`
2. 点击选择本地视频文件，或直接选择文件夹
3. 也可以把本地文件或文件夹直接拖进页面
4. 可以单独控制每个视频，也可以使用顶部工具栏进行批量操作

## 浏览器说明

- 文件夹导入依赖 `webkitdirectory`，更适合 Chromium / Edge
- 文件夹拖拽递归读取也依赖 Chromium 相关接口
- 画中画和部分媒体行为在不同浏览器里可能存在差异
